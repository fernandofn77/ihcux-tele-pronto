# TelePronto — Protótipo de Baixa Fidelidade

## Sobre o Projeto

O TelePronto é um aplicativo de telemedicina criado para reduzir filas em hospitais e facilitar o atendimento remoto para casos leves e moderados.

O sistema permite:

* Triagem inteligente
* Consulta por vídeo
* Receitas digitais
* Alarmes de medicação
* Busca de farmácias próximas

---

# Análise de Acessibilidade

O protótipo foi desenvolvido pensando principalmente em usuários idosos ou em situação de mal-estar.

Foram aplicadas as seguintes soluções:

* Botões grandes e fáceis de clicar
* Textos com alta legibilidade
* Poucos elementos por tela
* Contraste elevado
* Navegação simplificada
* Ícones intuitivos
* Informações principais destacadas
* Botão de emergência sempre visível

Essas decisões ajudam usuários com:

* visão turva
* tremores nas mãos
* ansiedade
* dificuldade de concentração

---

# Fluxo Crítico

O fluxo principal para iniciar uma consulta é:

Home → Triagem → Sala de Espera → Vídeo-Chamada → Receita Digital

O usuário consegue iniciar o atendimento em poucos passos, reduzindo o tempo necessário para receber ajuda médica.

---

# Prevenção de Erros

Para evitar problemas durante a consulta:

* O botão “Encerrar Consulta” possui confirmação antes de finalizar.
* Os botões principais possuem tamanho ampliado.
* As informações médicas aparecem de forma clara e organizada.
* O sistema mostra mensagens de status durante o carregamento e espera.
* O botão de emergência permanece disponível em todas as telas.
